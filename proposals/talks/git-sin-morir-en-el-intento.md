# Title

Git sin morir en el intento

## Duration

40 minutes - 1 hour

## Abstract

### Spanish

Git es nuestro pan de cada día, pero... a veces parece que no sabemos lo que estamos comiendo. Veremos el modelo mental de git de una manera diferente y cercana para ayudarte a entender por qué funcionan las cosas como funcionan y que la herramienta que usas día a día se convierta en tu mayor aliada.

Commits, ramas, merge, rebase, pull, push, fetch. Sé la persona que mejor conoce git de tu equipo, y por el camino le perderás miedo a los comandos.
